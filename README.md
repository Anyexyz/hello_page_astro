# Astro Portal

基于 Astro 重制的导航站点


## 食用指南

你需要关注 src/*.yml 文件，这些文件定义了站点的内容、分类和列表

site.ts 定义了站点相关的配置

图标基于iconfont

需要自定义图标，登录iconfont网站自建项目即可
